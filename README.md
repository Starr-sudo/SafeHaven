# SafeHaven

SafeHaven is a secure, AI-assisted mental health app with:

- An anonymous community feed (mood posts, support reactions, comments)
- A context-aware AI chat companion (emotion/topic-aware supportive conversation)

It is designed for privacy-first local storage and easy backend integration later.

## Repository Layout

- `frontend/` — React + Vite frontend app
- `backend/` — Backend APIs and services

## Deployment

See [DEPLOYMENT.md](./DEPLOYMENT.md) for step-by-step instructions on how to deploy your own fork of SafeHaven to Vercel (no repository ownership required).
